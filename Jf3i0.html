3. 변수


자료형   변수의 메모리 공간의 특징을 표현

변수   변수값 은 메모리 공간          과 변수값으로 구분되고, 변수값은 다양한 값으로 변경 가능

예)  12  =  5  +  7  에서 왼쪽 변수는 연산결과를 저장할 공간을 나타내고, 오른쪽 변수는 변수값을 의미

final   변수 공간에 변하지 않는 상수값을 저장한다고 선언하는 키워드

기본자료형   변수의 메모리공간에 변수값을 저장하는 자료형

    var           100        

참조자료형   변수의 메모리공간에 객체의 주소를 저장하는 자료형

    var   (Integer객체01)       (Integer객체01)   100   등의 데이터와 메서드 포함

자동형변환   자료형마다 저장방식이 다르므로, 이항연산이 가능하도록 시스템에서 자동으로 자료형을 변환
예) 10.0 / 4 = 10.0 / 4.0 = 0.25, int pi = 3.141592 = 3, char alphbet = 97 = 'a'

강제형변환   프로그래머가 명시적으로 자료형을 변환

(double)   값의 자료형을 강제로 실수 자료형으로 형변환
예) 10 / 3 = 3이지만, ((double)10) / 3 = 3.333333

(int)   값의 자료형을 강제로 정수 자료형으로 형변환
예) (int)3.14 = 3 또는 (int)'A' = 65

(char)   값의 자료형을 강제로 문자 자료형으로 형변환
예) (char)65 = 'A' 또는 (char)97 = 'a'
