알고리즘

 프로그램 시작 
    년도와 달을 입력 
    해당달의 시작요일과 말일을 계산하기 
    달력을 출력 
 프로그램 종료 
 
 해당달의 시작요일을 계산하는 함수 
    1년1월1일부터 직전년도 12월31일까지 전체 날짜를 계산 
    해당년도 1월1일부터 직전달 말일까지 날짜 계산 
    해당달 1일이 일요일부터 월요일 사이에 어느 요일인지 계산 
 
 해당달의 말일을 구하는 함수 
    해당달이 2월이고 윤년이면, 29일 
    아니면 월별 말일(31, 28, 31, 30, ... ) 값 적용 
 
 달력을 출력하는 함수 
    해당년도와 해당달을 출력 
    1일 이전은 공백 출력 
    1일부터 말일까지 
       일주일을 한줄에 출력 
       즉, 토요일 후 일요일은 새 줄에서
