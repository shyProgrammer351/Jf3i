    문제

    국어 점수, 영어 점수, 수학 점수를 입력받아 평균을 계산하는 문제입니다 이를 해결하는 다음 프로그램의 실행상태를 확인하세요 
    0점 ~ 100점 사이의 국어 점수를 입력하세요: 90
0점 ~ 100점 사이의 영어 점수를 입력하세요: 89
0점 ~ 100점 사이의 수학 점수를 입력하세요: 86
국어 90점, 영어 89점, 수학 86점의 평균은 88.33점입니다.

    프로그램 코드

    	import java.util.Scanner;
	
	public class Score
	{
1		public static void main( String[] args ) {
			Scanner scan = new Scanner( System.in );
			int korean = 0, english = 0, mathematics = 0;								
2			double average = 0.0;								
			
3			System.out.print( "0점 ~ 100점 사이의 국어 점수를 입력하세요: " );
4			korean = scan.nextInt();
5			System.out.print( "0점 ~ 100점 사이의 영어 점수를 입력하세요: " );
6			english = scan.nextInt();
7			System.out.print( "0점 ~ 100점 사이의 수학 점수를 입력하세요: " );
8			mathematics = scan.nextInt();
			
9			average = (double)( korean + english + mathematics ) / 3;			
			
10			System.out.printf( "국어 %d점, 영어 %d점, 수학 %d점의 평균은 %.2f점입니다.", 		
								korean, english, mathematics, average );
			
11		}
	}
